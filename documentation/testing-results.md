# Testing Results

This document contains the testing results for the Basic SOC / Security Monitoring Lab.

## Test Summary

| Test | Event ID | Expected Result | Actual Result |
|---|---|---|---|
| Failed login attempt | 4625 | Failed logon event appears in Event Viewer | Success |
| Successful login | 4624 | Successful logon event appears in Event Viewer | Success |
| User account created | 4720 | User creation event appears in Security logs | Success |
| Account lockout | 4740 | Account lockout event appears after multiple failed attempts | Success |
| Account unlock | N/A | Locked account is unlocked in Active Directory | Success |

A failed login attempt was generated by entering an incorrect password. The event was detected using Event ID 4625.

## User Account Creation Test

A new domain user account was created in Active Directory Users and Computers. The activity was detected using Event ID 4720.
