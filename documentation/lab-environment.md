# Lab Environment

## Overview
This lab uses a virtual Windows domain environment to practice basic security monitoring and event analysis. The focus is on identifying Windows security events that are commonly reviewed by IT support and entry-level SOC analysts.

## Virtual Machines

| Device | Role | Operating System |
|---|---|---|
| Server2022 | Domain Controller | Windows Server 2022 |
| Desktop1 | Domain Client | Windows 10 |
| Kali-Linux | Optional Testing Machine | Kali Linux |

## Domain Information

| Item | Value |
|---|---|
| Domain Name | SimoTech.com |
| Domain Controller | server2022.simotech.com |
| DNS Server | 192.168.10.10 |
| Client Machine | Desktop1 |

## Purpose
The purpose of this lab is to practice reviewing Windows security logs, identifying failed and successful login attempts, and documenting security findings in a simple incident report format.
