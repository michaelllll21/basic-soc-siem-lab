# Security Event IDs

This document lists common Windows Security event IDs reviewed in this lab.

| Event ID | Event Name | Description |
|---|---|---|
| 4624 | Successful Logon | A user successfully logged on to the system |
| 4625 | Failed Logon | A user failed to log on to the system |
| 4634 | Logoff | A user logged off from the system |
| 4720 | User Account Created | A new user account was created |
| 4725 | User Account Disabled | A user account was disabled |
| 4726 | User Account Deleted | A user account was deleted |
| 4740 | Account Locked Out | A user account was locked out |
| 4732 | Member Added to Security Group | A user was added to a security-enabled local group |

## Notes
These event IDs are useful for basic security monitoring, account activity review, and incident investigation.
